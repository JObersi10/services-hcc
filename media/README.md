## Format notes
- Recommended: square (1:1) for pfp/review images, landscape (e.g. 4:3 or 16:9) for work images.
- JPG works, but PNG/WebP also work as long as you keep the exact filename + extension above.
- If a file isn't present, the existing placeholder (initials/icon + gradient) stays as-is — nothing breaks.
